1. Enable CSP with restrictive policy in `security.middleware.js`
2. Hash OTPs before database storage in password reset flow
3. Remove College Admin credentials from API response; implement secure delivery

### Phase 2 (Critical - P1 Security)
4. Add rate limiting to `/refresh` and `/change-password` endpoints
5. Standardize password policy (8+ chars with complexity) across all user types
6. Fix HOD security findById() bypass in timetable controllers
7. Fix HOD security findById() bypass in timetableSlot controllers
8. Fix HOD security findById() bypass in timetableException controllers

### Phase 3 (High - P1 Data Integrity)

10. Implement proper User.isActive scoping on suspend/restore
